# .WIM-Manager
Utilidad script-paghettizada altamente editable para instalar Windows mediante imágenes .ESD/.WIM por red/PXE (sólo necesitas TFTP, SMB y DHCP). Es capaz de diferenciar entre equipos (UUID de la placa) por "whitelist", ofreciendo en función de esto diferentes imágenes y opciones de postinstalación. Hecho con amor y apto para scriptkiddies como yo.
